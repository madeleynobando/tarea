¿Cómo se podrá usar y manejar está página web de publicidad?

    Los dueños de negocios o emprendimientos:

  • 1. Ingresar a esta página web informatva
  • 2. Dar click en el enlace que colocaremos aqui una vez que aprendamos a manejar una base de datos
  • 3. Cuando demos click en el enlace nos direccionara a otra página web donde podremos ingresar en la parte de inicio la informacion de nuestro emprendimiento.
  • 4. Debemos ingresar datos como fotos del emprendimiento, direccion, nombre del emprendimiento, servicios o productos que ofrece
  • 5. Daremos click en "Registrar emprendimiento"
  • 6. En un lapso de 24 horas se subirá la informacion a la página web de forma que sea llamativa para los potenciales clientes


Ir al Formulario

    La persona que maneja la página web:

  • 1. Se registrarán los datos automáticamente en una base de datos
  • 2. Cuando ya tenga la información la subire en la página web
  • 3. Se enviará un mensaje al dueño del negocio notificandole que su página se esta publicitando correctamente
  • 4. Subiré el link del sitio web a redes sociales para que mas clientes puedan ver los negocios.
